Aviso: se uso la version 11.0.20.1 de javac para compilar los archivos.

Datos de interes:
    - A la clase zona y sus respectivos hijos se le agregaron 2 metodos
      nuevos, que solo ayudan a la ambientación del juego y no afecta al
      funcionamiento del juego como tal.

Instrucciones de compilacion y ejecución:

    -  Para compilar se debe encontrar los siguientes archivos en la carpeta:
        Amarillo.java, Cyan.java, Enemigo.java, Ilevantable.java, Juego.java, 
        Magenta.java, Muralla.java, Pieza.java, Pikinim.java, Pildora.java, Zona.java,
        makefile, README.txt
    -  Usar el comando "make run" para que se compilen todos los archivos y este mismo
       ejecutara el programa.
    -  Para borrar los archivos .class generados en la compilación, puede usar la linea
        de comando "make clean"
